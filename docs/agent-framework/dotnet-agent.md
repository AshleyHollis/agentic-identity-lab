# Agentic Layer — .NET

> **Path:** `apps/agent-gateway/dotnet-agent-framework/`  
> **Terminology:** This service is the lab's **Agentic Layer** — the application-level agent
> orchestration component. The directory is named `agent-gateway` for historical reasons (see ADR 0006).

This directory is reserved for the .NET Agentic Layer implementation. For now, the Python FastAPI
implementation is the runnable reference while the .NET service scaffolding is planned.
